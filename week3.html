OBJECT METHOD: this

Functions that are stored in object properties are called “methods”. Methods allow objects to act. Methods can reference the object as this.

OBJECTS

An object is a set of related values and functions. If a property is a function it will be called a METHOD. An object literal is an object that is created directly in the language by wrapping all its properties and methods in curly braces.

Create an object by entering a pair of curly braces. Don't use constructor function.

Access objects properties:

  • dot notation
  • superman.name
  • Bracket notation
  • superman[‘name’]

superman["real" + " " + "name"] // the property is built using string concatenation "Clark Kent"

To call an object’s method we can also use dot or bracket. superman.fly() superman['fly']()
